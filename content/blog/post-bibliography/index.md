> [!TIP]
> **Библиография - это не просто список источников в конце статьи.** Это способ показать, на чьих плечах вы стоите, и дать читателю дорожную карту для погружения в тему.
 
- **Академическая честность** – Чужие идеи должны быть атрибутированы. Плагиат убивает репутацию.
- **Воспроизводимость** – Читатель может найти и проверить ваши источники.
- **Профессиональный рост** – Хорошая библиография показывает, что вы в курсе современных исследований.
- **Автоматизация** – Правильные инструменты избавляют от ручного форматирования ссылок.
 
## Основные форматы библиографических ссылок
 
| Формат | Где используется | Пример оформления |
| :--- | :--- | :--- |
| **ГОСТ Р 7.0.100-2018** | Россия, СНГ (дипломы, диссертации, статьи) | Пыхтеева М.И. Название книги. М.: Наука, 2024. 300 с. |
| **APA (7th ed.)** | Психология, педагогика, социология | Smith, J. (2024). Title of the book. Publisher. |
| **MLA (9th ed.)** | Филология, литературоведение, искусство | Smith, John. *Title of the Book*. Publisher, 2024. |
| **Chicago / Turabian** | История, искусство, издательское дело | Smith, John. *Title of the Book*. Chicago: Publisher, 2024. |
| **IEEE** | Технические науки, IT, инженерия | [1] J. Smith, "Title of article," *Journal Name*, vol. 10, no. 2, pp. 10-20, 2024. |
| **BibTeX / BibLaTeX** | LaTeX, Overleaf, научные статьи | `@book{smith2024, author={John Smith}, title={Title}, year={2024}}` |
 
## Инструменты для управления библиографией
 
| Инструмент | Платформа | Особенность |
| :--- | :--- | :--- |
| **Zotero** | Win/Mac/Linux | Бесплатный, открытый код, отличные плагины для браузера и Word/LO |
| **Mendeley** | Win/Mac/Linux | Бесплатный, хорошая социальная сеть учёных, PDF-аннотации |
| **EndNote** | Win/Mac | Платный, мощный, стандарт в крупных университетах |
| **JabRef** | Java (везде) | Бесплатный, лучший для работы с BibTeX/BibLaTeX |
| **Paperpile** | Веб, Google Docs | Платный по подписке, идеален для экосистемы Google |
 
> [!NOTE]
> **Zotero — золотой стандарт для начинающих.** Бесплатно, просто, есть всё необходимое. Синхронизация между компьютерами, плагин в браузер (одним кликом сохраняет статью с сайта), экспорт в любой формат.
