> [!TIP]
> **Легковесные языки разметки — это простой текст, который превращается в красивый документ.** Никаких мышек, панелей инструментов и бесконечных «сохранить как...». Пишешь — и сразу видишь результат.

- **Человекопонятный формат** – Файл `.md` можно открыть в блокноте и всё равно понять структуру.
- **Без привязки к вендору** – Word есть не у всех. Markdown читается везде: от GitHub до обычного терминала.
- **Версионность из коробки** – Это обычный текст. Git отлично показывает, что изменилось в абзаце. С бинарным `.docx` так не работает.
- **Экспорт во что угодно** – Из одного исходника получаете HTML, PDF, DOCX, презентацию и даже сайт.

## Самые популярные языки этой группы

| Язык | Где используется | Особенность |
| :--- | :--- | :--- |
| **Markdown** | GitHub, заметки, README, форумы | Абсолютный стандарт индустрии |
| **reStructuredText** | Документация Python (Sphinx) | Мощнее Markdown, но сложнее |
| **AsciiDoc** | Техническая документация | Подходит для больших книг и мануалов |
| **Org-mode** | Emacs, планирование задач | Встроенная система GTD и таблиц |
| **Textile** | Старые блоги и CMS | Был популярен в 2010-х |
